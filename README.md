## What Is MIMZY?

MIMZY is a quantum workbench that unites ten thoroughly reviewed instruments for investigating quantum computing concepts. Inspired by the computational legacy of Lovelace, Babbage, and Turing, this platform simulates everything from elementary quantum gates to complete cryptographic protocols. It is designed for students, researchers, and enthusiasts seeking practical exposure to quantum algorithms without requiring physical quantum hardware.

What distinguishes MIMZY is its distinctive narrative: the workbench is presented as rediscovered futuristic technology from the past — a UD0 sphere concept that fuses historical computing heritage with contemporary quantum simulation. The environment includes a fully operational Antikythera Mechanism simulator alongside modern quantum tools, creating an unconventional link between ancient computational devices and quantum information science.

## Key Instruments

- **Antikythera Mechanism Simulator** — Examine the ancient astronomical calculator reinterpreted within a quantum context
- **Quantum Primer** — Guided introduction to foundational quantum computing concepts
- **Bloch Lab** — Interactively visualize qubit states on the Bloch sphere
- **Circuit Simulator** — Design and execute quantum circuits using standard gate libraries
- **Two-Qubit Lab** — Investigate entanglement and two-qubit operations
- **Error Correction** — Explore quantum error correction codes and their dynamics
- **Quantum Wavefield** — Observe wave function evolution across simulated quantum systems
- **BB84 QKD** — Implement the BB84 quantum key distribution protocol
- **E91 Entanglement Keys** — Work with the E91 protocol using entangled photon pairs
- **Observatory** — Monitor and analyze quantum system measurements
- **Quantum Dots** — Simulate quantum dot behavior and energy level structures
- **Reading Room** — Browse curated quantum computing literature and documentation

## Getting Started

Clone the repository to your local environment:

```bash
git clone https://github.com/tylerx82/mimzy-bb84-e91-grover.git
cd mimzy-quantum-workbench
```

Because MIMZY runs as a web application, simply open the `index.html` file in any modern browser. No build tools or server configuration are needed.

## How to Use

Launch MIMZY by opening the main HTML file in your browser. The interface presents a dashboard from which you can access any of the ten quantum instruments. Each instrument operates independently, enabling you to:

1. Choose an instrument from the main menu
2. Adjust parameters using the provided controls
3. Run simulations and view results in real time
4. Export data or save your work for later sessions

For the BB84 and E91 protocols, follow the guided workflow to simulate key exchange between parties. The Circuit Simulator accepts standard quantum gate notation and displays measurement outcomes.

## Configuration

MIMZY stores user preferences and session data locally in your browser using the Web Storage API. You can modify simulation parameters, display settings, and default instrument configurations through the settings panel accessible from the main dashboard. No external configuration files are required.
